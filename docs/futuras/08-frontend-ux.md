# Frontend UX - Configuracao de SLA e Automacao

## Menu alvo
- Configuracoes
  - SLA
  - Templates
- Automacao
  - Regras (Workflow)
  - Logs

## Tela SLA
- tempos (FRT, reply, resolucao, pending, fila)
- automacoes (auto-close, reabertura, lembretes)
- horario comercial
- prioridade (baixa/normal/alta/urgente)

## Tela Regras (Workflow)
- lista de regras (nome, trigger, ativo)
- filtro por trigger/canal/fila
- editor de regra por blocos
- preview do JSON gerado

## Tela Logs
- regra executada
- status (success/error/skipped)
- ticket associado
- timestamp
- erro detalhado

## Requisitos de UX
- alterar status de ativo/inativo sem perder historico
- clonar regra
- versionar regra
- exibir impacto estimado antes de publicar
